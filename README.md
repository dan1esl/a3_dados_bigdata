# A3 Análise de Dados e Big Data
Projeto de Análise de dados com contextualização na área de Esports

## Objetivo
Tangibilidade do conhecimento através de um produto real e foco em resultados práticos e aplicabilidade imediata.

Foco em soluções eficazes e contextualizadas.

Análise crítica e seleção estratégica de dados.
